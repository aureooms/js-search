# Installation

Can be managed using
[yarn](https://yarnpkg.com/en/docs),
[npm](https://docs.npmjs.com),
or [jspm](https://jspm.org/docs).


### yarn
```terminal
yarn add @aureooms/js-search
```

### npm
```terminal
npm install @aureooms/js-search --save
```

### jspm
```terminal
jspm install npm:@aureooms/js-search
```
